## Third-party components

### Waveshare ScreenKey modules

ScreenKey modules (SKU 34168) are a
[Waveshare](https://www.waveshare.com/0.85inch-screenkey.htm?sku=34168)
product. Open Screen Deck documents how to integrate them but does not include
Waveshare firmware, apps, or module internals.

This project is **not affiliated with or endorsed by Waveshare**.

### Elgato Stream Deck

Open Screen Deck is inspired by the per-key LCD macro pad use case popularized
by Elgato Stream Deck. This project is **not affiliated with or endorsed by
Elgato**, and "Stream Deck" is Elgato's trademark.
