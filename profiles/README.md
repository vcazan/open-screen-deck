# Profile Gallery

Community-shared deck layouts. Every `.osdprofile.json` here can be
imported straight into the companion app: **Profiles → Import**.

Profiles are self-contained — key configs, host actions, and (optionally)
key images/animations travel in one file. Export your own with the
**Share** button on any profile card.

## Contribute

1. Design your deck, then **Profiles → Share** to export the file
2. Rename it something descriptive (`obs-podcast-setup.osdprofile.json`)
3. Open a PR adding it here with a one-line description below
